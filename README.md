# Vue News App

> Practicing Test Driven Design in Vue JS with a clone of the Hacker News site.

## Tech and Dependencies

- Vue JS
- Testing with Jest

## Project setup

```
npm install
npm run serve
```

- Compiles and minifies for production
```
npm run build
```

- Lints and fixes files
```
npm run lint
```
